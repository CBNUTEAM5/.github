## 🚀 Projects

로드밸런서를 이용한 HTTP 웹서버 구축

## 🎯 Our goals

- 성능 향상: 다수의 클라이언트 요청을 효율적으로 처리하기 위한 최적화된 로드밸런싱 알고리즘 설계
- 확장성: 수평 확장을 통해 서버의 처리 용량 증가
- 안정성: 부하 분산으로 서버 과부하를 방지하고 안정적인 요청 처리 보장
